Selamat Datang!

Aplikasi ini memungkinkan Anda mengirim dan menerima berkas atau situs web ke/dari gawai "Windows 10/11".

Penyesuaian - Android

  • Harap aktifkan bluetooth!
  • Izin lokasi diperlukan untuk mencari gawai terdekat.

Penyesuaian - Windows

  • Silakan ikuti petunjuk di https://nearshare.shortdev.de/docs/setup pada gawai Windows Anda.

Mengirim

  • Anda dapat mengirim berkas dengan menggunakan fitur berbagi bawaan android, atau gunakan tombol di bawah.
  • Anda akan melihat notifikasi di WinOS tentang penerimaan berkas.

Menerima

  • Anda dapat menerima berkas dengan mengklik tombol di bawah atau menggunakan ubin cepat (quick settings).
